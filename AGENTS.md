# Project instructions for Codex

This is a WordPress project for a personal old-school gym journal and long-form blog.

Editorial north star:
“Just write like an intelligent man leaving notes in the margins of modern life.”

Tone:
Serious, reflective, dryly humorous, old-school, human. Avoid influencer language, SaaS language, hype, fake motivation and generic fitness content.

Design:
Dark iron-era gym atmosphere. Think 1990s bodybuilding VHS, Dorian Yates / Blood & Guts, training notebooks, chalk, old plates, quiet discipline. Use restraint.

Code:
Keep WordPress code clean, semantic, accessible and performant. Avoid unnecessary dependencies. Follow existing repo conventions first.

Content pillars:
- 46-year-old dad cutting from about 22% body fat
- Road to 100 kg bench press
- Road to 1000 lb club

Preferred recurring weekly sections:
- Current Stats
- This Week in Training
- Food / Nutrition Notes
- Mental Notes
- Work / Modern Life Notes
- Iron Archive

Always preserve readability and maintainability over visual gimmicks.
